# Side Quest App - Frontend

Simple HTML/CSS/JavaScript frontend for testing the Side Quest App.

## How to Use

1. Make sure the backend is running:
   ```bash
   cd backend
   python app.py
   ```

2. Open `index.html` in your browser:
   - Double-click the file, OR
   - Right-click → Open with → Your browser

3. Test the features:
   - Login with any username/password (creates account automatically)
   - Add sidequests with titles and descriptions
   - Mark sidequests as complete
   - Add friends by their username
   - View your sidequests and friends lists

## Features

- 🔐 Login / Sign Up
- ➕ Add Sidequests
- ✅ Complete Sidequests
- 👥 Add Friends
- 📋 View all your quests and friends
- 🎨 Beautiful gradient design
- 📱 Mobile responsive

## Note

Make sure the backend server is running on `http://localhost:5000` before using the frontend!
